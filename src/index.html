    Editer l'animation

    Aide

    Placez toutes vos images dans le répertoire img.
    Si l'une d'elles se nomme background.jpg, elle sera utilisée comme décor de fond à mettre derrière les animations. Les images relatives aux animations peuvent prendre le nom qu'on veut.
    Il faut ensuite créer des fichiers d'animation dont l'extension doit être .json. En voici un exemple : 
          {
      "time": 600,
      "frames": [
      {"src":"monsterA-walk-1.png","y":48,"x":29},
      {"src":"monsterA-walk-2.png","y":48,"x":29},
      {"src":"monsterA-walk-3.png","y":48,"x":34},
      {"src":"monsterA-walk-4.png","y":48,"x":30}
      ]
      }
    L'attribut time indique la durée totale d'un cycle d'animation, exprimée en millisecondes.
    Chaque frame est une image composant l'animation. Les coordonnées x et y sont celles du point d'ancrage de l'image. A l'aide de l'interface, il faut essayer de faire correspondre ce point d'ancrage avec l'intersection des lignes en pointillés.
